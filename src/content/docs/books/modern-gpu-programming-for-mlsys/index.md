# Modern GPU Programming For MLSys

机器学习系统是现代 AI 工作负载的核心。在这些系统中，性能往往取决于少数几个 GPU 核函数（kernel）的质量。注意力核函数、LLM 预填充与解码核函数、低精度块缩放 GEMM（通用矩阵乘法）、融合的 MoE 层以及其他大型融合核函数，都直接影响着训练与服务两方面的端到端速度。

然而，要让这些核函数变快，我们需要的不仅仅是一系列优化技巧。现代 GPU 早已不再是旧设计的简单变体。近期的架构引入了更丰富的存储空间、新的访问模式以及日益特化的执行单元。要把它们编程好，我们既需要对硬件有清晰的心智模型，也需要对高性能核函数如何构建有实践层面的理解。本书正是围绕这两方面能力的培养而展开。

本书遵循一条简单的递进路线：先理解 GPU 硬件，再学习我们将要使用的编程模型，最后一步步构建出最先进的核函数。我们的主要目标是 Blackwell 架构，主要的贯穿示例是快速矩阵乘法（GEMM）和 Flash Attention。在过程中，我们还会研究 GPU 优化背后的核心要素：数据布局、异步数据移动以及异步协调。

这些材料源自卡内基梅隆大学 [Machine Learning Systems](https://mlsyscourse.org/) 课程系列。为了便于学习与运行，本书使用 **TIRx** Python DSL（领域特定语言）一步步构建真实的 GPU 核函数示例。TIRx 贴近硬件，使我们能在通过可运行代码学习的同时，对底层控制进行推理。

## 本书组织结构

- **第一部分，理解 GPU。** 本部分介绍 GPU 的整体组织结构、编写快速核函数的通用方法，以及数据布局、异步内存操作和协调等关键概念。它为本书其余部分所依赖的硬件直觉打下基础。
- **第二部分，TIRx 概览。** 本部分介绍 TIRx 的关键要素，它们是全书代码示例的基础。
- **第三部分，GEMM：从分块到 SOTA。** 一份完整的分块 GEMM 优化指南，通过 TMA 流水线、持久化调度、线程束特化以及 2-CTA（协作线程数组）集群逐步构建而成。
- **第四部分，Flash Attention 4。** 一个基于第三部分技术构建的完整注意力核函数：两个 MMA 之间穿插 softmax、在线 softmax 重缩放、因果掩码以及 GQA（分组查询注意力）。
- **参考。** TIRx 语言参考与编译器内部实现。
